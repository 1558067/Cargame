# Cargame
JavaScriptで作成した車ゲーム

# DEMO
 ![demo](https://github.com/1558067/Cargame/blob/main/demo.gif)
# Usage
【URL】https://1558067.github.io/Cargame/  
操作説明  
アクセル：「X」　バック：「Z」　車を前に傾ける：「→」　車を後ろに傾ける：「←」  

車が横転しないように、うまく操作して砂漠を駆け抜けろ！！
